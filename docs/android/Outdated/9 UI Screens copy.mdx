### Note About `UserItemClickListener`

To get click event of the list you must use `UserItemClickListener`. The `UserItemClickListener` has
`setItemClickListener()` on handle single click and `setItemLongClickListener` to handle long click events.

```java
CometChatUserListScreen.setItemClickListener(new UserItemClickListener() {
                @Override
                public void setItemClickListener(User user, int position) {
                    super.setItemClickListener(user, position);
                }

                @Override
                public void setItemLongClickListener(User user, int position) {
                    super.setItemLongClickListener(user, position);
                }
            });
```
